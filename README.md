## Requirements

This architecture depends on **RxSwift**, **RxCocoa** and **RxSwiftExt**.

## Installation

### Swift Package Manager

```
.package(url: "https://github.com/geppetto-ios/Geppetto.git", .upToNextMajor(from: "0.2.3"))
```

### CocoaPods

```
pod "Geppetto"
```

### Carthage

```
github "geppetto-ios/Geppetto"
```
